# reference, 참조자
```c++
int main(void)
{
	int a = 3;
	int& another_a = a;

	another_a = 5;
	std::cout << "a: " << a << std::endl;
	std::cout << "another_a: " << another_a << std::endl;

	return (0);
}
```
앰퍼샌드 연산자를 자료형에 붙이면 reference가 된다. reference는 포인터와 유사하지만 다른 개념이다. 포인터처럼 다른 변수(혹은 상수)를 가리키지만, 메모리상에 존재하지는 않는다.
